# TGVAE: Transformer Graph Variational Autoencoder for Generative Molecular Design 

## Installation 
- Python 3.7+

```bash
git clone https://github.com/nnhoangtrieu/TGVAE.git
pip install requirements.txt
```

--- 
## Usage 

### Training 
```bash
python train.py 
```


### Generate molecules
```bash
python generate.py 
```
